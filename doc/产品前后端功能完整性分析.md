### 三、🔴 发现的问题与风险

#### **严重问题 (P0)**

1. **后端访问页面未实现过期检查**
   - **问题**: `pub/{project_id}/index.html` 直接访问静态文件，没有PHP中间层检查过期状态
   - **影响**: 过期项目仍可被访问，失去限制意义
   - **当前方案**: 依赖`expire_cron.php`定时任务替换文件内容，但存在延迟
   - **建议**: 使用Nginx/Apache重写规则，将所有请求路由到PHP检查脚本

2. **免费用户5分钟过期限制可能被绕过**
   - **问题**: 后端虽然强制设置`expire_minutes=5`，但如果定时任务未执行，文件不会被替换
   - **影响**: 免费用户的项目可能永久有效
   - **建议**: 在访问时实时检查数据库过期时间

3. **多语言实现不完整**
   - **问题**: 代码中使用`.localized`扩展，但未找到`.strings`本地化文件
   - **影响**: 部分文本可能无法正确翻译
   - **位置**: 需要检查`LanguageManager.swift`的实现方式

#### **中等问题 (P1)**

4. **订阅状态同步不可靠**
   - **问题**: `SubscriptionManager`依赖本地`@AppStorage`存储Pro状态，卸载重装会丢失
   - **影响**: 已付费用户可能失去Pro权限
   - **建议**: 增加服务器端订阅验证作为后备

5. **运营后台数据存储**
   - **问题**: 筛选配置保存在`/tmp/`目录（虽然已迁移到`getDataDir()`）
   - **影响**: 某些服务器配置下可能丢失
   - **状态**: 部分已修复，需确认`getDataDir()`的实现

6. **访问日志IP匿名化**
   - **问题**: 数据库schema中`ip_address`存储完整IP，不符合GDPR
   - **影响**: 隐私合规风险
   - **建议**: 使用迁移脚本`migration_ip_anonymization.sql`

7. **App Store ID配置**
   - **问题**: `expired.php`中使用占位符`YOUR_APP_ID`
   - **影响**: 过期页面下载链接无效
   - **实际ID**: `6764022927`（在`AppConfig.swift`中）

   - 客户端重试机制有限

9. **性能优化空间**

**Nginx 配置亮点**：

1. **HTML 请求通过网关**：

   ```nginx
   location ~ ^/pub/([a-z0-9]+)/([^.]+\.(html|htm))$ {
       rewrite ^/pub/([a-z0-9]+)/([^.]+\.(html|htm))$ /index.php?project_id=$1&file=$2 last;
   ```

   ```nginx
   location ~* \.(css|js|png|jpg|jpeg|gif|svg|webp|...) {
       expires 7d;
       add_header Cache-Control "public";
       access_log off;
   }
   ```

3. **安全防护**：
   ```nginx
   location /data/ { deny all; }
   location /database/ { deny all; }
   location ~ \.(bak|backup|old|tmp)$ { deny all; }
   ```

1. 确认Nginx已重载：`sudo nginx -t && sudo systemctl reload nginx`
2. 清除浏览器缓存
3. 检查是否有其他location块冲突

**原因**：语法错误

**解决**：
1. 检查大括号 `{}` 是否配对
2. 检查正则表达式是否有特殊字符未转义

```php
function anonymizeIP($ip, $salt = null) {
    // 1. 生成SHA256哈希（前16位）
    $hash = substr(hash('sha256', $ip . '_' . $salt), 0, 16);
    
    // 2. IPv4匿名化（最后一段改为0）
    // 192.168.1.100 → 192.168.1.0
    $parts = explode('.', $ip);
    $parts[3] = '0';
    $anonymized = implode('.', $parts);
    
    return ['hash' => $hash, 'anonymized' => $anonymized];
}
```

✅ [projects.php:663](file:///Volumes/ssd/aicode_new0421/ioscode/zaixianhtml/deploy_package/api/projects.php#L663) - 显示日志时调用 `maskIP()`：

```php
function maskIP($ip) {
```
